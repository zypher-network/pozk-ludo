# PoZK shuffle prover contracts

```
npm run build # build all contracts
npm run abi   # build all ABI files in ../public/ABI
```

## Deploy
### Locally
```
npx hardhat node
npm run deploy
```

### Chains
```
npm run deploy --network xxx
```

## Test
`npm run test`
